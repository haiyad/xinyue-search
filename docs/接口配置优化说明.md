## 使用指南

### 测试线路

1. 进入接口配置页面
2. 找到要测试的线路
3. 点击"测试"按钮
4. 输入搜索关键词（默认"仙逆"）
5. 点击"开始测试"
6. 查看测试结果

### 复制线路

1. 进入接口配置页面
2. 找到要复制的线路
3. 点击"复制"按钮
4. 修改线路名称（自动添加"副本"后缀）
5. 修改其他配置（可选）
6. 点击"确认添加"

### 搜索线路

1. 进入接口配置页面
2. 在搜索框中输入关键词
3. 选择筛选条件（可选）
4. 点击"搜索"按钮
5. 查看筛选结果

### 批量删除

1. 进入接口配置页面
2. 勾选要删除的线路
3. 点击"批量删除"按钮
4. 确认删除操作

## 技术细节

### 前端技术

**Vue.js 组件**:
- 使用 Vue.js 2.x
- Element UI 组件库
- Axios HTTP 客户端

**关键方法**:
- `clickTest()`: 测试线路
- `clickCopy()`: 复制线路
- `getList()`: 获取线路列表
- `batchDelete()`: 批量删除
- `postRequest()`: 公共请求方法

### 后端技术

**PHP 框架**:
- ThinkPHP 6.x
- RESTful API 设计

**新增接口**:
- `POST /admin/api_list/batchDelete`: 批量删除
- `POST /admin/api_list/getList`: 支持筛选参数

**筛选参数**:
- `keyword`: 线路名称（模糊搜索）
- `type`: 线路类型（精确匹配）
- `pantype`: 网盘类型（精确匹配）
- `status`: 状态（精确匹配）

### SSE 实时通信

**测试功能使用 SSE**:
- 前端创建 EventSource 对象
- 监听 `onmessage` 事件
- 实时接收搜索结果
- 自动关闭连接

## 注意事项

1. **测试功能**:
   - 测试时会调用真实的搜索接口
   - 建议使用默认关键词测试
   - 测试结果仅供参考

2. **复制功能**:
   - 复制的线路默认禁用
   - 需要手动启用
   - 建议修改线路名称

3. **批量删除**:
   - 删除操作不可恢复
   - 建议先备份重要配置
   - 确认后再执行

4. **搜索筛选**:
   - 支持多条件组合筛选
   - 清空条件可显示全部
   - 搜索结果实时更新
