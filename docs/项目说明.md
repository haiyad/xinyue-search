# 网盘资源管理系统 - 项目说明

## 项目概述

网盘资源管理系统是一个支持多网盘转存分享的综合性资源管理与索引工具。系统支持夸克网盘、百度网盘、阿里云盘、UC网盘、迅雷云盘等多种网盘平台，提供资源管理、搜索、转存、分享等功能。

**当前版本**：v3.6  
**许可证**：MIT License

## 技术栈

### 后端框架
- **ThinkPHP 6.x** - PHP企业级开发框架
- **MySQL** - 关系型数据库

### 核心依赖
- **Lizhichao/Word** - 中文分词库（VicWord）
- **PHPExcel** - Excel文件处理
- **phpqrcode** - 二维码生成

### 网盘API集成
- 夸克网盘API
- 百度网盘API
- 阿里云盘API
- UC网盘API
- 迅雷云盘API
- 115网盘API
- 天翼云盘API
- 移动云盘API

## 核心功能模块

### 1. 资源管理模块

#### 资源CRUD操作
- **资源列表**：支持分页、筛选、排序
- **资源添加**：支持单个添加和批量导入
- **资源编辑**：修改资源信息
- **资源删除**：支持单个删除和批量删除

#### 批量导入功能
- 支持Excel文件导入（.xlsx, .xls）
- 自动去重：根据标题和网盘类型判断重复
- 自动生成关键词：通过百度搜索API生成资源关键词
- 支持字段：资源名称、资源地址、资源分类、添加人、关键字搜索、资源介绍

#### 资源转存功能
- **一键转存**：支持夸克、百度、阿里、UC等网盘转存
- **批量转存**：支持批量转存多个资源
- **转存分享**：转存后自动生成分享链接
- **广告过滤**：自动过滤转存资源中的广告文件

### 2. 搜索功能模块

#### 搜索模式
- **精准搜索**：完全匹配关键词
- **模糊搜索**：包含关键词即可
- **分词搜索**：使用中文分词库进行智能搜索

#### 搜索特性
- 支持标题和描述字段搜索
- 支持分类筛选
- 支持网盘类型筛选
- 搜索结果高亮显示
- 搜索权重排序

### 3. 全网搜索模块

#### 搜索源类型
- **API接口**：第三方API搜索源
- **TG频道**：Telegram频道搜索
- **网页爬虫**：网页爬取搜索

#### 搜索功能
- 支持多搜索线路配置
- 支持搜索线路权重设置
- 支持屏蔽关键词功能
- 支持自动跳转功能
- 支持资源有效性过滤

## 数据库设计

### 主要数据表

#### qf_source（资源表）
- source_id：资源ID（主键）
- source_category_id：资源分类ID
- title：资源名称
- description：资源描述/关键词
- url：资源地址
- code：提取码
- fid：文件ID
- is_type：网盘类型
- status：状态（0禁用 1启用）
- is_delete：是否删除（0否 1是）
- is_top：是否置顶
- sort：排序
- page_views：浏览量
- create_time：创建时间
- update_time：更新时间

#### qf_source_category（资源分类表）
- source_category_id：分类ID（主键）
- name：分类名称
- sort：排序
- status：状态

#### qf_user（用户表）
- user_id：用户ID（主键）
- username：用户名
- password：密码
- avatar：头像
- status：状态

#### qf_admin（管理员表）
- admin_id：管理员ID（主键）
- username：用户名
- password：密码
- role_id：角色ID

## API接口

### 前台API

#### 搜索接口
- `GET /api/search/index` - 获取资源列表
- `GET /api/search/getDetail` - 获取资源详情
- `GET /api/search/getNew` - 获取最新资源
- `GET /api/search/getHot` - 获取热门资源
- `GET /api/search/getCategory` - 获取分类列表

#### 工具接口
- `GET /api/tool/getConfig` - 获取系统配置
- `POST /api/tool/upload` - 上传图片
- `GET /api/tool/getAdsCode` - 获取广告位
- `POST /api/tool/feedback` - 用户反馈
- `GET /api/tool/ranking` - 获取排行榜
- `GET /api/tool/Qsearch` - 全网搜索

### 后台API

#### 资源管理接口
- `GET /admin/source/getList` - 获取资源列表
- `POST /admin/source/add` - 添加资源
- `POST /admin/source/update` - 修改资源
- `POST /admin/source/delete` - 删除资源
- `POST /admin/source/imports` - 批量导入
- `GET /admin/source/excel` - 导出Excel
- `POST /admin/source/transfer` - 一键转存
- `POST /admin/source/transferAll` - 全部转存
- `GET /admin/source/getFiles` - 获取网盘文件
- `POST /admin/source/batchUpdateCategory` - 批量修改分类
- `GET /admin/source/generateKeywords` - 生成关键词

## 安装部署

### 环境要求
- PHP >= 7.4
- MySQL >= 5.7
- Apache/Nginx
- 推荐使用宝塔面板或LNMP环境

### 安装步骤
1. 下载项目源码
2. 上传到服务器网站目录
3. 访问/install进入安装向导
4. 按照向导完成配置
5. 删除install目录

### 配置说明
- 数据库配置：config/database.php
- 应用配置：config/app.php
- 网盘Cookie：后台系统配置中设置
